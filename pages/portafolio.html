Esta es nuestra área de desarrollo web
En Web Dev transformamos ideas en presencia digital. Diseñamos y desarrollamos páginas web personalizadas para que negocios, profesionistas y creadores de contenido tengan una forma profesional de mostrar quiénes son, qué hacen y cómo pueden ser contactados.

Creamos desde páginas tipo link-in-bio para centralizar enlaces y páginas que funcionan como tu portafolio digital, hasta landing pages para negocios o pequeños emprendedores completamente personalizadas.

Información importante y consideraciónes

- ¿Dónde se aloja la pagina?

usamos un servicio gratuito de hosting de github pages
por lo cual el dominio o URL de las páginas web termina en .github.io
por ejemplo: mipaginaweb.github.io/

- Dominio/URL personalizado.

podemos cambiar el dominio o URL a uno personalizado (.com, .com.mx, .org, .net, etc...)
sin embargo, es un servicio adicional que requiere renovación anual si se quiere conservar el dominio personalizado

- Soporte y actualizaciónes

se realizan actualizaciónes de la pagina web como servicio adicional, contamos con planes de actualización y soporte si estos serán constantes.

- cambios mayores

si se requiere un rediseño o rebranding completo de la pagina, se considera como un trabajo adicional.
no se incluye en los planes de actualización

Página web tipo Link-in-Bio


Este tipo de página web es ideal para creadores de contenido que buscan centralizar todos sus enlaces en un solo lugar.
Redes sociales, canales, tiendas, plataformas, formularios de contacto y cualquier otro enlace importante pueden reunirse en una página personalizada diseñada para representar tu marca o identidad como creador.
Además, al tratarse de una página propia, tienes mayor control sobre su diseño, contenido y funcionalidades.

Página web como portafolio digital


¿Eres ilustrador, músico de sesión, profesionista independiente o prestas algún servicio de forma individual?
Si buscas una manera más profesional de mostrar tu trabajo, proyectos, experiencia y formas de contacto en un solo lugar, una página web tipo portafolio digital puede ser la solución ideal.
Comparte tu trabajo con clientes potenciales mediante una página diseñada para destacar lo que haces y facilitar que las personas conozcan tus servicios.
